# Documentation Directory

This directory contains all project documentation organized by category.

## Directory Structure

### `development/`
Development-related documentation:
- `DEVELOPMENT_PLAN.md` - Overall development roadmap
- `PROGRESS_UPDATE_EMAIL.md` - Progress updates
- `README_PAPERSCRAPER.md` - Paperscraper integration guide

### `integration/`
Technical integration documentation:
- Source-specific integration guides
- Implementation details
- Testing procedures

### `policies/`
Legal and policy documentation:
- Website scraping policies
- Terms of service compliance

### `project/`
Project-level documentation:
- `deploy-cloud.md` - Cloud deployment guide
- `REPORT.md` - Project reports

### `reports/`
Analysis and status reports:
- Database issue reports
- Performance analysis
- Status assessments
